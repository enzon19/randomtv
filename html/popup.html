RandomTV

Pick a random item from a list on Trakt.

DISPLAY
Choose what happens after the item is picked.
ADVANCED
Show which method was used to pick the item.

API: Uses Trakt's API to get items in lists with more than one page. Can't be used when: have filters that hide items or the list is private.
Local: Uses a script in the browser to get a item. Used when: the list have just one page, the server fails, have filters that hide items or the list is private.
